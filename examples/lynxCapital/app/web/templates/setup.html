Automate setup

Use the in-repo scripts to provision Lynx providers, resources, and the baseline policy. The scripts use their built-in Caracal defaults, so only the zone and scoped control key need to be exported.

1. Create the zone

Create or select the Lynx zone in Caracal Console. Use that zone id in the export block below.

2. Create a scoped control key

In the Control menu, create a key with only the scopes below. Console returns the one-time client id and client secret.

{% for scope in setup_automate.scopes %}{{ scope }}{% endfor %}

3. Run provision

Run the script from the Lynx Capital directory. It reads the catalog, creates providers and resources, and publishes the baseline policy.

Required environment

export CARACAL_ZONE_ID="zone_lynxcapital"
export CONTROL_CLIENT_ID="app_lynx_bootstrap"
export CONTROL_CLIENT_SECRET="one-time-secret-from-console"

Run scripts

python scripts/provision.py
python scripts/teardown.py
