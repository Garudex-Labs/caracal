Automate setup

1. Create the zone

Go to Caracal Console > Zones > New. Set name = "Lynx Capital". One zone backs the whole platform.

2. Create the control key

Go to Control > control key create. Set name = "Lynx Capital Bootstrap", max token TTL = 300, and expires in days = 30.

3. Export credentials and run

Copy the returned client_id and client_secret.

Add to examples/lynxCapital/.env.provision

CONTROL_CLIENT_ID="{{ setup_env.controlClient }}"
CONTROL_CLIENT_SECRET="{{ setup_env.controlSecret }}"

Provisioning credentials stay in .env.provision, separate from the workload .env. The script idempotently creates the applications, providers, resource views, and policy set, then prints the per-application credential exports.
