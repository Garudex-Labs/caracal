Tutorials sit between [Get Started](/get-started/) and the surface-by-surface [Guides](/guides/). Each tutorial names an outcome you want to deliver and walks through the Caracal surfaces that produce it.

| Outcome | What you do | Surfaces touched |
| --- | --- | --- |
| [Protect an API](./protect-an-api/) | Put a Caracal-enforced boundary in front of a service you own. | Console (resource, policy, grant), SDK connector or Gateway routing. |
| [Connect an agent](./connect-an-agent/) | Wire an agent app through the SDK so its tool calls are authority-checked and audited. | TS / Python / Go SDK `Caracal.connect()`, Gateway. |
| [Inspect a run](./inspect-a-run/) | Trace one agent run end-to-end through authorization decisions and action results. | Console (sessions, agents, audit), `audit explain`. |

If you do not yet have a running stack, start at [Quickstart: Run the Stack](/get-started/quickstart/). If you want surface-by-surface reference, jump to [Guides](/guides/) or [Concepts](/concepts/).
