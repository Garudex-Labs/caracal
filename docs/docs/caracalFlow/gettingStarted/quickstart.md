# Caracal Flow Quickstart

Get Caracal Flow running in under a minute.

## Prerequisites

- Caracal Core installed and running.
- Python 3.10+

## Launch

```bash
caracal flow
```

This launches the TUI. Use arrow keys to navigate and `Enter` to select.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate menus |
| `Enter` | Select |
| `q` | Quit |
| `?` | Help |
