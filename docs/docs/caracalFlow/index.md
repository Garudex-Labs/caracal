# Caracal Flow

Caracal Flow is a terminal-based user interface (TUI) for managing your Caracal deployment. It provides a rich, interactive experience for:

- **Agent Management**: Register, view, and configure agents.
- **Policy Configuration**: Create and edit spending policies in real-time.
- **Spending Dashboard**: Monitor agent spending and budget utilization.
- **Quick Setup**: Guided wizards for initial configuration.
