# Getting Started with Caracal Enterprise

Caracal Enterprise is the centralized control plane for managing authority enforcement across your organization.

## Overview

Caracal Enterprise provides a web dashboard and API for:

- **Define Authority** -- Create policies that govern what your principals can do.
- **Enforce Rules** -- Use the Caracal Gateway or SDK to enforce mandates in real-time.
- **Audit Activity** -- Track every mandate issuance, validation, and denial in an immutable ledger.

## The Dashboard

The Enterprise Dashboard is your command center:

| Section | Description |
|---------|-------------|
| **Principals** | Register and manage AI agents, users, and services |
| **Policies** | Create and update authority policies |
| **Mandates** | Issue, view, and revoke execution mandates |
| **Analytics** | Real-time authority event monitoring and audit logs |
| **Gateways** | Manage deployed Caracal Gateway instances |
