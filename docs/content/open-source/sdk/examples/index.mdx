These examples show the public SDK entry points used by both canonical SDKs.

**Audience:** Open-source SDK users and integrators

## Python Tool Call

```python
from caracal_sdk import CaracalClient

client = CaracalClient(api_key="sk_test_123")

try:
    result = await client.tools.call(
        tool_id="provider:github:resource:issues:action:create",
        tool_args={"title": "Investigate regression"},
        metadata={"request_id": "req-123"},
    )
finally:
    client.close()
```

## TypeScript Tool Call

```ts
import { CaracalClient } from "@caracal/core";

const client = new CaracalClient({ apiKey: "sk_test_123" });

try {
  const result = await client.tools.call({
    toolId: "provider:github:resource:issues:action:create",
    toolArgs: { title: "Investigate regression" },
    metadata: { request_id: "req-123" },
  });
} finally {
  client.close();
}
```

## Workspace-Scoped Call

Python:

```python
ctx = client.context.checkout(workspace_id="ws_1")
result = await ctx.tools.call(
    tool_id="provider:slack:resource:messages:action:post",
    tool_args={"channel": "alerts", "text": "Runtime check complete"},
)
```

TypeScript:

```ts
const ctx = client.context.checkout({ workspaceId: "ws_1" });
const result = await ctx.tools.call({
  toolId: "provider:slack:resource:messages:action:post",
  toolArgs: { channel: "alerts", text: "Runtime check complete" },
});
```

## Builder Initialization

Use the builder when setup needs an explicit base URL, custom transport, or extensions.

Python:

```python
from caracal_sdk import CaracalBuilder

client = (
    CaracalBuilder()
    .set_api_key("sk_test_123")
    .set_base_url("http://localhost:8000")
    .build()
)
```

TypeScript:

```ts
import { CaracalBuilder } from "@caracal/core";

const client = new CaracalBuilder()
  .setApiKey("sk_test_123")
  .setBaseUrl("http://localhost:8000")
  .build();
```
