# First Flow Session

## Outcome

At the end of this tutorial, you will complete the `first flow session` flow and verify expected runtime behavior at each step.

## Prerequisites

- Access to a host environment with required runtime dependencies for Caracal.
- A configured workspace context for the target operation.
- Permission to run CLI and runtime actions in your environment.

## Walkthrough

1. Prepare the runtime context and confirm required environment values are present.
2. Execute the first action for this flow and check command output for successful initialization.
3. Continue through the core operation sequence and validate each transition before proceeding.
4. Capture resulting identifiers and verify they are reusable in the next step.
5. Run a final verification step to confirm end-to-end behavior in the same session.

## Verification checklist

- Expected command outputs are present with no runtime errors.
- State changes are visible in the corresponding CLI or API read path.
- Subsequent commands can reuse produced identifiers without reconciliation steps.
