# CLI

## Scope

This page maps the CLI command groups used in the open-source runtime. Use it as the entry point before executing command-specific procedures.

## Command groups

- [Host Commands](/open-source/end-users/cli/host-commands) - host orchestration and runtime lifecycle commands.
- [Workspace](/open-source/end-users/cli/workspace) - workspace selection and lifecycle commands.
- [Principal](/open-source/end-users/cli/principal) - principal identity management commands.
- [Policy](/open-source/end-users/cli/policy) - policy authoring and validation commands.
- [Authority](/open-source/end-users/cli/authority) - authority evaluation and status commands.
- [Delegation](/open-source/end-users/cli/delegation) - delegation creation/revocation commands.
- [Audit](/open-source/end-users/cli/audit) - audit and trace inspection commands.
- [Provider](/open-source/end-users/cli/provider) - provider registration/control commands.
- [Doctor](/open-source/end-users/cli/doctor) - diagnostics and readiness checks.

## Execution order guidance

1. Prepare runtime with host commands.
2. Establish workspace/principal/policy state.
3. Execute authority/delegation operations.
4. Verify outcomes in audit and doctor paths.
